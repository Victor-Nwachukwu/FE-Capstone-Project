About the Project
Knowledge Quest is a dynamic and interactive trivia quiz application designed to test your knowledge on a variety of subjects. Built with React.js and styled using Tailwind CSS, the app provides a seamless and visually appealing experience on both desktop and mobile devices.
The application features a modern, responsive design and a simple, user-friendly interface. With a focus on a smooth user journey, Knowledge Quest is the perfect platform for quick, challenging trivia sessions.

Features
* Welcome Page: A clean and engaging splash page to greet users.
* Login Page: A secure entry point for users to begin their quiz journey.
* Quiz Category Selection: Users can choose from different quiz categories to customize their experience.
* Quiz Page: A dynamic quiz interface with a built-in 30-second timer for each question to add a layer of challenge.
* Result Page: A summary page that displays the user's final score and performance.
* Responsive Design: The application is fully responsive and optimized for a great experience on any screen size, from mobile phones to large desktops.

Technologies Used
* React.js: A JavaScript library for building user interfaces.
* Tailwind CSS: A utility-first CSS framework for rapid styling.
* Vercel: The platform used for seamless deployment and hosting.

Getting Started
To get a local copy up and running, follow these simple steps.
Prerequisites:
* Node.js (LTS version recommended)
* npm or yarn

Installation:
1. Clone the repository: git clone https://github.com/Victor-Nwachukwu/FE-Capstone-Project.git
2. Navigate to the project directory: cd FE-Capstone-Project
3. Install dependencies: npm install (or yarn install)
4. Run the development server: npm start (or yarn start)
The app will be available at http://localhost:5173.
